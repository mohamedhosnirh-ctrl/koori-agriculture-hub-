# 🌾 koori — Modern Farm Marketplace

koori is a modern, responsive web platform designed for farmers to browse, add, and manage agricultural products such as animals, medications, seeds, tools, and more. The project focuses on simplicity, speed, and a clean user experience inspired by natural farm colors and friendly UI animations.

---

## ⭐ Features

- Multi‑Language Support — English, Arabic, French, Italian
- Category‑Based Navigation — Each category opens a dedicated page
- Dynamic Cart System — Add items to cart with LocalStorage persistence
- Countdown Offers — A timer synced across pages
- Responsive Design — Fully optimized for all devices
- Clean UI & Animations — Smooth hover effects and modern layout
- Independent Category Pages — Each category displays all related products

## 🧠 How It Works

### Category Cards
Each category on the home page is represented as a clickable card. When clicked, it redirects to a dedicated page (e.g., cows.html).

### Category Pages
Each page displays all products related to that category. Products can be added to the cart.

### Cart System
Items are stored in LocalStorage, allowing persistence across pages.

### Language System
All text elements use data-translate attributes. Language changes instantly without reloading the page.
